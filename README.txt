# 記憶小遊戲

這是一個簡單的記憶力訓練遊戲，玩家需記住隨機生成的字母或數字按鍵，並在限定時間內輸入正確答案以提高記憶力。

## 遊戲流程

1. **開始遊戲**：
   - 畫面上有一個「開始遊戲」按鈕，按下後進入準備階段。

2. **準備階段**：
   - 隨機選擇三個按鍵（字母或數字）顯示為綠色，玩家有 5 秒記住按鍵。

3. **玩家階段**：
   - 玩家需在 5 秒內按下記住的按鍵。
   - 正確按鍵會變成綠色，錯誤按鍵變為紅色。

4. **遊戲結束**：
   - 顯示結果 MessageBox，並根據猜對或猜錯顯示不同顏色。
   - 點擊確定後重新開始遊戲。

## 使用提示

- 36 個按鍵可以使用 Button Array 實現。
- 重設遊戲時使用 `Controls.Remove()` 來清除按鍵狀態。
- 輸入的按鍵建議轉換成大寫以方便判斷。
